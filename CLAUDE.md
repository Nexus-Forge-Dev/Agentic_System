# Forge Nexus — Agent System

## How This System Works

This project uses a 6-division multi-agent system. The **Orchestrator** (Chief of Staff) plans, routes, and reviews all work. Each division has a **Lead** and **Specialists** who execute tasks within their domain. The Research Council handles high-stakes adversarial decisions.

Every skill runs an 8-step pre-flight: read MANIFEST → read rules → read PROJECT → read learned patterns → verify task → validate permissions → scope side-effects → log to audit. Nothing executes before the pre-flight passes.
